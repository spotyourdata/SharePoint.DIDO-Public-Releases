# SharePoint.DIDO — Public Releases

> Business Central app by [SpotYourData](https://www.spotyourdata.nl)

This repository contains the public runtime releases of the **SharePoint.DIDO** app for Microsoft Dynamics 365 Business Central.

---

## About

**SharePoint.DIDO** integrates Microsoft SharePoint directly with Business Central, enabling seamless document management and collaboration from within BC.

---

## Requirements

- Microsoft Dynamics 365 Business Central (v21 or higher)
- Microsoft SharePoint (Online or On-Premises)
- Valid SpotYourData license

## Installation

1. Download the latest `.app` file(s) from the [Releases](../../releases) page
2. Open Business Central and go to **Extension Management**
3. Click **Upload Extension** and select the downloaded `.app` file
4. Follow the installation wizard

## Releases

All releases are published on the [Releases](../../releases) page. Each release includes:

- Runtime `.app` files ready for installation
- Release notes with changes and fixes
